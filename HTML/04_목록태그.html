HTML/CSS/Javascript

HTML
웹페이지를 만들기 위한 표준 마크업 언어
HyperText Markup Language
CSS
웹 페이지의 스타일(색상, 레이아웃, 글꼴 등)을 지정하는 언어이다.
HTML 구조에 디자인을 적용하여 사용자 경험을 향상
Javascript
웹 페이지에서 동적인 기능(이벤트, 애니메이션, 데이터 변경 등)을 구현하는 프로그래밍 언어
